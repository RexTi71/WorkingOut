# WorkingOut

## About
Web app to keep track of your exercises for each day with user login

## Technologies used

**Spring Boot** - main framework for creating this project
**Htmx** - for simplified CRUD operations
**Thymeleaf** - for easy view rendering

## Usage

### Requirements
- Java SDK 25
- PostgreSQL

1. Create Database "workingout"
2. Create in **src/main/resources/** new file named: **"application-local.properties"**  
```#Change with your db configuration
DB_USERNAME=YOUR_DB_USERNAME
DB_PASSWORD=YOUR_DB_PASSWORD```
